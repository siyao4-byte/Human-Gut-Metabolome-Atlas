# Data handling

Do not copy the large annotation and abundance collections into this OneDrive workspace.

- Keep external source data read-only.
- Store paper-specific metadata extracts in `metadata/`.
- Store rebuildable merged data in `intermediate/`.
- Store analysis-ready datasets in `processed/`.
- Record exclusions and merge decisions in `../outputs/qc/`.

The first main processed output will be:

`processed/paper2_total_list.csv`
